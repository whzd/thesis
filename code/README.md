# Code

This folder contains the two developed components of the solution, namely the API and the WebApp.

## Content

The [api](https://github.com/whzd/thesis/tree/master/code/api) folder contains the api developed in [Flask](https://flask.palletsprojects.com/en/1.1.x/) and is responsible for generating the explanation of a given word.

The [website](https://github.com/whzd/thesis/tree/master/code/website) folder contains the webapp developed in [React](https://reactjs.org/) and it allows the user to interact with the developed API.
